# Eclipse Flip

A 3D puzzle game about restoring balance.

**[Play Online](https://krjadhav.github.io/eclipse_flip/)**

## How to Play
*   **Goal:** Create a "Total Eclipse" by turning all tiles **Black**.
*   **Controls:** Drag to select square areas (2x2, 3x3, etc.).
*   **Mechanic:** Selected tiles flip colors (White â†” Black).

## Tech Stack
*   React + Vite
*   Three.js (@react-three/fiber)
*   Zustand (State Management)
*   Framer Motion
